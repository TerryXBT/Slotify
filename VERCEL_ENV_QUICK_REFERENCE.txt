================================================================
快速操作步骤
================================================================

1. 打开 https://vercel.com/login
2. Add New → Project
3. 导入 Slotify 仓库
4. 展开 Environment Variables
5. 逐个添加上述 9-11 个变量
6. 点击 Deploy
7. 等待构建完成
8. 更新 NEXT_PUBLIC_APP_URL 为实际 URL
9. 重新部署

预计时间: 10-15 分钟
